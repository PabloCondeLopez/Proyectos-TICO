Capa de contenido: documento HTML

En esta parte, que esta compuesta por el documento HTML, se incluye el contenido y su estructuración semántica. No se especifica absolutamente nada relacionado con la presentación, como puede ser el color escogido para las fuentes, el fondo de la página o la tipografía y tamaño de la misma, entre muchas otras cosas.

Capa de presentación: hoja de estilo CSS

En esta segunda capa se define el aspecto del contenido introducido en el documento HTML. Se pueden aplicar hojas de estilo para diferentes dispositivos de visualización, como pantallas, impresoras, PDA, etc. De esta forma se optimiza el aspecto del sitio web para atendiendo a las necesidades específicas de cada dispositivo.

Capa de comportamiento: Javascript

Esta capa se destina para funcionalidades con más complejidad, como despegables o rollovers. Esta capa interactúa con el DOM (Document Object Model). El código Javascript deberá estar en contenido en archivos externos, así como las hojas de estilo CSS. Se pueden usar otros codigos como CGI o PHP, aunque Javascript es el usado en la vasta mayoría de los casos.
